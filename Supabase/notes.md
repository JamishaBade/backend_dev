# Supabase
It is a BaaS. (Backend as a service)
It gives database, authentication, sotrage for files and other tools (All without you needing to ser up a server from scratch.)

- It sets up the backend for you.
### Database (PostgreSQL): 
- Store your app's data (users, products, posts, etc)
- Comes with an easy dashboard to view/ edit data.
### Authentication
- Users signup & login with email/passowrd
- Social logins like google
### Storage
- upload and manage images, videos and files
- Example: Profile pictures, product images, PDFs
### Real-time updates 
- get live data updates without refreshing the page.
- Example: a chat app that shows new messages instantly. 

## BTS of Supabase
- it is built on topof PostgreSQL
- it adds extra tools on top of PostgreSQL like auth, file storage, and APIs.
