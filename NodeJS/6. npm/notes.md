# Node Package Manager:
It is the default package manager for node.js

Dependency Management: Handles external libraries (packages) your project relies on.
Versioning Control: Ensures reproducible builds using semantic versioning (semver).
Script Automation: Lets you define project-level scripts (like start, test, build) in package.json.
Ecosystem Access: Grants instant access to hundreds of thousands of libraries via the npm registry.


## npm init
initializes package.json
package.json is the metadata blueprint of your project: name, version, entry point, scripts, dependencies, author, license.

## installing packages
npm install packagename
npm uninstall packagename
